# FILM!

## Установка

1. Скопировать на сервер:
  - файлы окружения,
  - папку public,
  - docker-compose
  - конфиги ngnix
2. развернуть контейнеры  
3. Заполнить БД
4. подключить сертификат. 

Для упрощения копирования и разворачивания можно воспользоваться файлами в директории bin
